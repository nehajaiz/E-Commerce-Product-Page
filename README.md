# E-Commerce-Product-Page
A responsive e-commerce product page built using HTML, CSS, and JavaScript with dynamic product listing, category &amp; price filtering, add-to-cart functionality, quantity management, LocalStorage-based cart persistence, and modern UI interactions.

🚀 Features

Dynamic product listing using JavaScript
Product filter by category and price range
Sorting (price low → high, high → low)
Add to cart with quantity update
Prevent duplicate cart items
Cart data saved using LocalStorage
Product details modal
Wishlist functionality
Responsive design (mobile-first)

🛠️ Tech Stack

HTML5

CSS3 (Flexbox, Grid, Media Queries)

JavaScript (ES6+)


🎯 Learning Outcomes

JavaScript array methods (map, filter, sort)
DOM manipulation & event handling
LocalStorage for data persistence
Responsive UI design
Clean and modular frontend code
